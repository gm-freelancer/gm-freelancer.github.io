Lists and Formating

<ol>...</ol>

Create numbered (ordered) lists showing sequential order preference or priority.

<ul>...</ul>

Display a bulleted (unordered) list without any extra emphasis on order of importance.

<li>...</li>

Specifies each list item to be bulleted or numbered.

<dl>...</dl>

Used for definitions for list items sepecifically.

<dt>..</dt>

The definition of a single term inline with body content.

<dd>...</dd>

The discription for the specified term.

The dl tag is used in conjunction with dt (defines terms/names) and dd (describes each term/name).

Example:

<!DOCTYPE html> <html> <head> <title>My first website</title> </head> <body> <ol> <li>January</li> <li>February</li> <li>March</li> </ol> <ul> <li>Pakistan</li> <li>China</li> <li>AJK</li> </ul> <dl> <dt>MUST</dt> <dd>Located in Mirpur AJK</dd> <dt>NUML</dt> <dd>Located in Islamabad</dd> </dl> </body> </html>
