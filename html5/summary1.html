Document Summary

<html>....</html>

The tag shows up at the beginning and ending of an HTML document (known as the root element). It indicates that the webpage is written in HTML5, and all other page markup comes in between these beginning and ending tags.

<head>....</head>

It contains information that specific page, including the title tags, meta data , and links to scripts and style sheets.

<title>....</title>

The tag is the title of that page, useful for both search engines (when they can scan and index pages) and users (showing up in a browser's title bar) by explicitly stating the primary topic of each page.

<body>....</body>

Body tags include all content that will be shown to users including everything they'll see and read.

Example:

<!DOCTYPE html> <html> <head> <title>My first website</title> </head> <body> </body> </html>
