Links and Formating

<a href="">...</a>

Anchor tag for hyperlink.

<a name="name">...</a>

An anchor tag that is useful for bringing the user to a specific document element.

<a href="#name">...</a>

An anchor link that brings users specifically to a div element.

<a href="" target="">...</a>

An anchor tag that sets the designation of the link that is being click.
target="" takes the following values:
_blank Opens the linked document in a new window or tab
_self Opens the linked document in the same frame as it was clicked (this is default)
_parent Opens the linked document in the parent frame
_top Opens the linked document in the full body of the window
framename Opens the linked document in a named frame.

<a href="" download></a>

Directs the browser to download the linked resource rather than opening it.

Example:

<!DOCTYPE html> <html> <head> <title>My first website</title> </head> <body> <a href="gm-freelancer.github.io" target="_blank">Facebook</a> <a href="/wp-content/uploads/flamingo.jpg" download> Download this picture</a> </body> </html>
