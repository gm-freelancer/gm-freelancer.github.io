<img>

An image tag to include and display image files.
It can contains the following attributes:

src="url"

The url or file of the image to display.

alt="text"

Alternative text that helps explain the image content to both text engines and users.

height=""

It specifies the image height in pixels or percentages.

width=""

It specify the width of the image in pixels or percentages.

To make this work copy this code and the image you want to use in the same folder then just change the source link.
For example your image name is "must.jpg" just change the src="images/social.jpg" to src="must.jpg".

Example:

<!DOCTYPE html> <html> <head> <title>My first website</title> </head> <body> <img src="images/social.jpg" width="500px" height="500px;" alt="social_site"> </body> </html>
