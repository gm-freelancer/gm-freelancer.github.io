DOCUMENT STRUCTURE

<h1>...</h1> <h2>...</h2> . . . <h6>...</h6>

All six tags of Headings, with 1 being most important on a page and 6 being the last. These elements are used to describe content sections on a page.

<div>...</div>

A generic container used to denote a page section. This tag is often used to stylize other HTML elements with CSS or to perform certain tasks with JavaScript.

<span>...</span>

An inline section or block container, typically used for grouping styling elements.

<p>...</p>

This tag is used to organize paragraphs of text.

<br>

Creates a line break (or old-schools carriage-return), useful for writing blocks of text that need to be on different lines (think addresses, etc.)

<hr>

Creates a horizontal rule, a sectional break in an HTML page. Typically used to denote a change in topic or section of a page.

Example:

<!DOCTYPE html> <html> <head> <title>My first website</title> </head> <body> <div> <h1>This is my first web page.</h1> <p>I used a DIV tag, a PARAGRAPH tag, <span>a SPAN tag</span> , <br> a BR tag and a HR tag </p> <hr> <h2>I just used a HR tag.</h2> <p>The HR tag creates a line between the paragraphs.</p> </div> </body> </html>
